### How to build ASCII-Piano

`cd ..\..\ASCII-Piano`

`cmake -G "MinGW Makefiles" -B "build" -S "src"`

`cd build` go to build directory

`make` or `mingw32-make` for example

#### Execution
```
cd build
./letGo.exe
```
